To change admin password, open a terminal
Go to the webServer folder (PATH on the Iot server VM : /home/iot-server/server/webServer/)
Run this command : node new_admin_password.js
You need to write the current admin password
After you write the new password
the hash of the password will be display on the terminal
You must copy paste in the .env file (which is in the same folder /webServer/) at the line adminPassword (after the = symbol)

To change the admin login, you can change it in the same .env file at the line adminLogin (after the = symbol)
